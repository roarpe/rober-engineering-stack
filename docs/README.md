# Documentation

Este directorio sera el indice de la documentacion de ROBER ENGINEERING STACK.

La documentacion crecera de forma progresiva: solo se crearan carpetas y documentos cuando exista contenido real que mantener. No se usaran carpetas vacias, placeholders innecesarios ni documentacion ficticia para simular avance.

## Categorias previstas

### architecture

Documentacion arquitectonica estable del stack: decisiones de diseno, estructuras, reglas de composicion y vistas tecnicas.

La arquitectura vigente esta actualmente en [../ARCHITECTURE.md](../ARCHITECTURE.md).

### decisions

Registro futuro de ADRs, es decir, decisiones arquitectonicas importantes con contexto, alternativas y consecuencias.

Indice actual: [decisions/README.md](decisions/README.md).

### guides

Guias operativas para usar el stack: seleccion de modulos, ejecucion de gates, uso de agentes, verificacion y mantenimiento.

Estas guias se crearan en fases posteriores cuando exista contenido operativo real.
